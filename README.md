# 🐧 Guía Dual Boot: Ubuntu 26.04 LTS + Windows 11

> Guía gratuita y open-source para la comunidad de desarrolladores hispanohablantes.

## 📖 ¿Qué es esta guía?

Esta guía te lleva paso a paso a través de la instalación de **Ubuntu 26.04 LTS** en dual boot con **Windows 11**, optimizado para un stack de desarrollo de alto rendimiento.

No es solo una instalación de Linux. Es una configuración completa pensada para desarrolladores que quieren:

- 🎮 **Gaming en Windows 11** sin compromisos
- 💻 **Desarrollo full-stack en Ubuntu** con GNOME
- 🧠 **AI local ligero** con Ollama, Whisper y modelos locales
- 🐳 **Docker + NVIDIA Container Toolkit** para GPU en contenedores
- ⚡ **Rendimiento máximo** con drivers NVIDIA propietarios, CUDA y optimizaciones del kernel

## 🎯 ¿Qué vas a conseguir?

Al seguir esta guía tendrás:

| Área | Resultado |
|------|-----------|
| **Sistema** | Ubuntu 26.04 LTS + Windows 11 en dual boot, GRUB configurado |
| **GPU** | RTX 4060 con drivers NVIDIA propietarios + CUDA toolkit |
| **Dev Stack** | Python + uv, Node.js + pnpm, Go, Docker con soporte GPU |
| **AI Local** | Ollama con modelos locales, Faster-Whisper, LiteLLM proxy |
| **Productividad** | VS Code, herramientas CLI modernas (bat, eza, zoxide, fzf) |
| **Respaldo** | Snapshots con Timeshift para rollback instantáneo |
